# Chat Server Application

## 📌 Description
A real-time chat server application that allows multiple users to connect and communicate simultaneously using socket programming.

## 🛠️ Technologies Used
- Java
- Socket Programming
- Multithreading

## ✨ Features
- Multiple client connections
- Real-time messaging
- Thread-based client handling
- Server-client architecture

## ⚙️ How It Works
- Server listens on a port
- Clients connect to server
- Each client handled in separate thread
- Messages broadcasted to all users

## 🚀 How to Run
1. Run Server.java
2. Run Client.java 
3. Start chatting
